VPC ( Virtual private Cloud )

What is VPC :- 

VPC is a logically isolated virtual network in AWS that allows us to securely deploy cloud resources with complete control over networking, security, routing, and connectivity. It enables scalable, secure, and production-grade architectures.

CIDR ( Classless Inter-Domain Routing )
It is a method to define IP address ranges in networking and cloud platforms like AWS VPC, Kubernetes, and on-prem networks.

Before CIDR, IPs were divided into fixed classes (A, B, C).
This caused IP wastage and poor scalability.

CIDR solves this by:

Allowing flexible IP ranges

Efficient IP utilization

Better network planning

For every subnet, AWS reserves 5 IP addresses: Network address, VPC router,Future use,Broadcast

Subnet:-
VPC = Big network
Subnet = Smaller pieces of that network

Each subnet:

Belongs to one VPC

Exists in one Availability Zone (AZ)

Uses a portion of the VPC CIDR
