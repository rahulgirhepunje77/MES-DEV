Add Extra volume to Ec2 instance 

High-Level Steps

Create a new EBS volume (AZ must match EC2 AZ Availability Zone: same AZ as EC2)

Attach the volume to EC2 ( Device name: /dev/sdf (AWS name) )

Detect the disk in OS ( lsblk )

Create filesystem ( sudo mkfs -t ext4 /dev/xvdf )

Mount the volume ( sudo mkdir /data , sudo mount /dev/xvdf /data , df -h )

Make it persistent ( blkid /dev/xvdf (UUID will get) , sudo vi /etc/fstab , UUID="abc-123-xyz"  /data  ext4  defaults,nofail  0  2 )

test 

sudo mount -a

Done! Volume Added Successfully
